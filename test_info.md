### Название
Линейная рекуррентная последовательность: значение, сумма и взвешенная сумма

### Дополнительно
Задача хорошо подходит под решения на Python и C++, при этом требует не лобового моделирования, а эффективного алгоритма. Типичное корректное решение будет строиться на матричном быстром возведении в степень и обычно занимает примерно 100-300 строк в зависимости от стиля реализации.

### Формулировка
```
Задана линейная рекуррентная последовательность порядка k.

Известны:

первые k членов последовательности F1, F2, ..., Fk
коэффициенты c1, c2, ..., ck
Для любого n > k последовательность определяется формулой:

Fn = c1 * F(n-1) + c2 * F(n-2) + ... + ck * F(n-k)

Требуется для заданного n вычислить:

Fn — n-й член последовательности
Sn = F1 + F2 + ... + Fn
Wn = 1*F1 + 2*F2 + ... + n*Fn
Так как значения могут быть очень большими, все вычисления и ответы нужно выводить по модулю:

MOD = 1000000007

Формат входных данных
Первая строка содержит два целых числа:
k n

Вторая строка содержит k целых чисел:
c1 c2 ... ck

Третья строка содержит k целых чисел:
F1 F2 ... Fk

Формат выходных данных
Выведите три целых числа через пробел:

Fn Sn Wn

Все значения должны быть выведены по модулю 1000000007.

Ограничения

1 <= k <= 30
1 <= n <= 10^18
0 <= ci <= 10^9
0 <= Fi <= 10^9
```
